# Little Knight Adventure v0.5

一款使用 Godot 4.5 引擎开发的2D像素风格冒险游戏。

## 🎮 游戏特色

### 核心系统
- **玩家控制**: 流畅的移动、跳跃和攻击系统
- **状态机**: 实现了空闲、跑动、跳跃等状态切换
- **生命值系统**: 完整的玩家生命值管理
- **关卡系统**: 多个精心设计的关卡场景

### 商店交易系统
- 随机商品生成（每次3个商品）
- 多种道具类型（生命药水、魔法药水、增益效果等）
- 完整的金币系统和购买验证
- 自动保存系统集成
- NPC交互界面（E键打开商店）

### 敌人系统
- **蝙蝠**: 飞行敌人AI
- **史莱姆**: 地面敌人（普通和Boss版本）
- **Boss战**: 史莱姆Purple Boss

### UI系统
- HUD显示（生命值、金币等）
- 聊天气泡系统
- 商店UI（商品展示、购买、通知）

## 🚀 快速开始

### 要求
- Godot 4.5 或更高版本

### 运行游戏
1. 克隆或下载此仓库
2. 使用 Godot 4.5+ 打开项目
3. 点击"运行"按钮或按 F5

## 🎯 技术栈

- **引擎**: Godot 4.5
- **语言**: GDScript
- **架构模式**:
  - 状态模式 (State Pattern) - 玩家状态机
  - 单例模式 (Singleton) - 全局数据管理
  - 观察者模式 (Observer) - 信号系统

## 🎮 游戏操作

### 玩家控制
- **WASD / 方向键**: 移动
- **Space**: 跳跃
- **E**: 交互（打开商店等）
- **ESC**: 关闭商店

### 商店操作
- **鼠标**: 选择商品
- **点击**: 购买商品
- **ESC**: 关闭商店
